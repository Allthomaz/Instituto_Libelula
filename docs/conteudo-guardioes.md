## Como enviar as fotografias

Coloque os arquivos originais em:

```text
assets/images/guardioes/originais/
```

Use o nome indicado no campo `Arquivo da foto`. JPG, JPEG e PNG são aceitos como originais. A conversão para WebP será feita durante a integração dos assets.
